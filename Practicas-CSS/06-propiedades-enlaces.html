Propiedades de Enlaces en CSS

Los 4 Estados de los Enlaces

Los enlaces tienen 4 estados diferentes que podemos personalizar:

  • a:link - un enlace normal no visitado
  • a:visited - un enlace que el usuario ha visitado
  • a:hover - un enlace cuando el usuario pasa el ratón sobre él
  • a:active - un enlace en el momento en que se hace clic
IMPORTANTE: El orden debe ser: link - visited - hover - active (LVHA) para que funcione correctamente.
