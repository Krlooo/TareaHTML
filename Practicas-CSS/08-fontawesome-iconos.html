FontAwesome - Iconos como Fuentes

¿Qué es FontAwesome?

FontAwesome es una librería de iconos que se pueden usar como fuentes en tu página web. Ventajas:

  • Miles de iconos disponibles
  • Se pueden cambiar de tamaño fácilmente con CSS
  • Se pueden colorear como texto normal
  • Son escalables (vectoriales)

Cómo Incluir FontAwesome

1. Registrarse en fontawesome.com

Obtén un código kit personalizado

2. Añadir el script en el head <script src="https://kit.fontawesome.com/tu_codigo.js"></script>
3. Usar los iconos <i class="fas fa-nombre-icono"></i>
