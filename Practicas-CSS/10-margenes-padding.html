Márgenes (margin) y Relleno (padding)

Diferencia entre Margin y Padding

MARGIN (espacio exterior)
BORDER (borde)
PADDING (espacio interior)
CONTENT (contenido)

Margin: Espacio FUERA del elemento (entre elementos)
Padding: Espacio DENTRO del elemento (entre el borde y el contenido)

Propiedades de Margin

margin-top: 50px
margin-bottom: 30px
margin-right: 80px
margin-left: 60px

Margin con Método Abreviado

/* 4 valores: top right bottom left (sentido del reloj) */ margin: 25px 50px 75px 100px;
margin: 25px 50px 75px 100px
/* 2 valores: vertical horizontal */ margin: 20px 60px;
margin: 20px 60px (vertical | horizontal)
/* 1 valor: todos los lados iguales */ margin: 30px;
margin: 30px

Margin Auto (Centrar Horizontalmente)

Para centrar un elemento horizontalmente en su contenedor

margin: auto;
width: 300px;
¡Estoy centrado!

Propiedades de Padding

padding-top: 30px
padding-right: 40px
padding-bottom: 25px
padding-left: 50px

Padding con Método Abreviado

/* 4 valores: top right bottom left */ padding: 10px 20px 30px 40px;
padding: 10px 20px 30px 40px
/* 1 valor: todos los lados iguales */ padding: 25px;
padding: 25px en todos los lados

Width y Height NO incluyen Padding ni Margin

¡IMPORTANTE! Las propiedades width y height NO incluyen padding, bordes o márgenes. Solo establecen el tamaño del área de contenido.
width: 200px;
padding: 20px;
border: 5px;

Ancho total = 200 + 40 + 10 = 250px

El ancho total ocupado será: width + padding-left + padding-right + border-left + border-right
