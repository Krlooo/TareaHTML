Propiedades de Fuentes en CSS

font-family (Familias de Fuentes)

Esta fuente es Arial (sans-serif)

Esta fuente es Times New Roman (serif)

Esta fuente es Courier (monospace)

Esta fuente es Verdana con fallbacks: Arial, Helvetica, sans-serif

Fallbacks: Se recomienda poner al final la familia genérica de la fuente por si no se encuentra la principal.

Google Fonts

Fuentes importadas desde fonts.google.com

Fuente Roboto de Google Fonts - moderna y legible

Fuente Playfair Display - elegante para títulos

Fuente Lobster - decorativa y divertida

Fuente Courier Prime - monoespaciada de Google

Ventaja: Las fuentes de Google se descargan automáticamente, funcionan aunque el usuario no las tenga instaladas.

font-size (Tamaño)

Tamaño 14px (píxeles)

Tamaño 24px

Tamaño 1.5em (1.5 veces el tamaño base)

Tamaño 2em (el doble)

W3C recomienda: Usar unidades relativas como "em" en lugar de píxeles para mejor accesibilidad.

font-weight (Grosor)

Texto ligero (300)

Texto normal (400 o normal)

Texto en negrita (700 o bold)

font-style (Estilo)

Texto normal (font-style: normal)

Texto en cursiva (font-style: italic)

Método Abreviado (font)

Se puede combinar todo en una sola línea

font: italic bold 18px/1.5 Georgia, serif;

font: [style] [weight] [size]/[line-height] [family];
