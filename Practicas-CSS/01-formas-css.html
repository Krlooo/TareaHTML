3 Formas de Aplicar CSS a HTML

1. Estilo en línea (inline)

Este párrafo tiene estilos aplicados directamente en la etiqueta HTML

Otro ejemplo con fondo amarillo

2. Hoja de estilo interna (en el head)
Este div usa una clase definida en la sección <style> del head
Los estilos internos se aplican a toda la página
3. Hoja de estilo externa (archivo .css)
Este div usa una clase definida en un archivo CSS externo

Los archivos CSS externos se pueden reutilizar en múltiples páginas

Recomendación: Lo mejor es usar hojas de estilo externas porque:
  • Se pueden reutilizar en varias páginas
  • El código queda más limpio y organizado
  • Es más fácil de mantener
